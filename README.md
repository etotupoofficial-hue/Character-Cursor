# Персонаж-курсор

Это небольшая демо-страница, где персонаж следит за курсором, имитирует печать и двигает ртом от микрофона.

## Как использовать своего персонажа

1. Замените файл `character.png` на изображение вашего персонажа.
   - Можно использовать `SVG`, `PNG` или `JPG`.
   - Если используете другой формат, обновите ссылку в `index.html` (атрибут `src` у тега `<img>`).
2. Запускайте демо через `launch.bat` (Windows).

## Запуск в Windows

Дважды кликните по `launch.bat`. Он запустит локальный сервер и откроет страницу в отдельном окне.

> Чтобы закрыть сервер, вернитесь в окно командной строки и нажмите `Ctrl + C`.
